# Титаник — предсказание выживания

# Описание
Проект предсказывает, выживет ли пассажир Титаника, используя нейросеть.

# Данные
- 'train.cs' - данные с ответами (Survived)
- 'test.csv' - данные без ответов

# Признаки
Все признаки преобразованы в бинарные (0/1):
- **Pclass** - 1 если третий класс
- **Sex_parser** - 1 если женщина
- **Young** - 1 если моложе среднего возраста
- **Familly** - 1 если мало родственников

# Модель
Нейросеть на TensorFlow/Keras:
- Dense(128, relu)
- Dense(64, relu)
- Dense(1, sigmoid)

# Результат
Точность на тестовых данных: **80-83%**

# Запуск
1. Скачать 'train.csv' и 'test.csv'
2. Запустить 'titanic.py'

# Файлы
- 'titanic.py' — основной код
